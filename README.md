### 功能方面

所有接口均来自qq音乐。[demo体验](https://haipingxiaotong.github.io/music/demo/)

- 线上地址没有歌词，歌词需要启动服务！！！
- 歌词显示
- 歌曲下载
- 歌曲列表记录


### 使用

```javascript
git clone https://github.com/HaipingXiaotong/music.git

npm install 

npm start

```
